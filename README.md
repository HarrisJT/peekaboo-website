# peekaboo-website

> Official website for Peekaboo Pediatric Therapy LLC in Denver, CO

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run watch

# build for production with minification
npm run build

# run the application
npm start

# remove and reinstall dependencies
npm run reinstall

# lint
npm run lint
```
